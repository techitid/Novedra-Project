What You'll Learn in “AI Foundation”

Comprehensive curriculum covering everything from fundamentals to advanced AI systems

1. What is AI?

  • - History and key concepts.
  • - Branches of AI.

2. Machine Learning (ML) Basics

  • - Types of ML: Supervised, Unsupervised, Reinforcement Learning.
  • - Building your first ML model using scikit-learn.

1. Introduction to Python

  • - Basics: Variables, Data Types, Loops, Functions, and Classes.
  • - Libraries: NumPy, pandas, and matplotlib.

2. Getting Started with Jupyter Notebook

  • - Installation and setup.
  • - Interactive programming and visualization.

3. Assignment

1. Prompt Engineering

  • - Crafting effective prompts for AI models.
  • - Examples using OpenAI APIs (e.g., ChatGPT).

2. Introduction to Large Language Models (LLMs)

  • - Understanding LLMs like GPT and their applications.
  • - Hands-on exercises with OpenAI’s APIs.

3. Vision-Language Models (VLMs) and Multimodal Models

  • - Introduction to models like CLIP and DALL·E.
  • - Applications of multimodal AI.

4. Assignment

1. LangChain Basics

  • - Setting up LangChain for LLM chaining tasks.
  • - Building custom applications with LangChain.

2. LangFlow Introduction

  • - Using LangFlow to visually design AI workflows.

3. Assignment

1. What are AI Agents?

  • - Architecture and applications.
  • - Developing an AI agent using LangChain or LangFlow.
  • - RAG: Combining retrieval mechanisms with generative models for accurate, context-aware responses.
  • - Demonstration of RAG-enhanced AI agents retrieving info from external sources.

2. Real-World Use Cases

  • - Example 1: Projects such as task automation or chatbots.
  • - Example 2: Build a RAG-enabled chatbot using LangChain or OpenAI.

Incorporating retrieval mechanisms like databases or document stores.

3. Assignment

1. Key Concepts in Computer Vision

  • - Image processing basics.
  • - Object detection and classification.

2. Computer Vision with OpenCV

  • - Hands-on projects using Python and OpenCV.

3. Assignment

1. Assignment Projects

  • - Students choose and build projects combining all the skills learned.
  • - Examples: Chatbot with computer vision features, AI workflow automation.

2. Best Practices and Next Steps

  • - Ethical considerations in AI.
  • - Career paths in AI and ML.
